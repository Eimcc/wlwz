# 武林外传网页 UI/UX 优化方案

为了提升网页的视觉效果和交互体验，我计划从视觉设计、动态交互和技术重构三个维度进行全面优化。

## 1. 视觉效果优化 (UI Enhancement)
*   **细腻的色彩与质感**：在保留原有的“武侠风”基础上，引入更现代的色彩系统。利用 Tailwind 的 `backdrop-blur` 实现毛玻璃效果，增强层级感。
*   **暗色模式支持**：完善已有的 `useTheme` 钩子，为网页添加“深夜客栈”主题，提供更舒适的夜间浏览体验。
*   **排版与间距**：优化中文字体展示（如 `ZCOOL XiaoWei`），调整全局间距和响应式布局，确保在各种设备上都有良好的阅读体验。

## 2. 交互体验升级 (UX & Interactions)
*   **引入 Framer Motion**：安装并利用 `framer-motion` 库，实现以下交互：
    *   **入场动画**：首页 Hero 区域的标题和按钮将以流畅的滑入/淡入效果呈现。
    *   **滚动展示**：页面向下滚动时，内容块将通过“滚动揭示”动画平滑出现。
    *   **微交互**：为角色卡片、按钮添加悬停（Hover）缩放和光影反馈，增加操作的趣味性。
*   **平滑页面切换**：利用 React Router 和 `AnimatePresence` 实现页面间的淡入淡出过渡。

## 3. 技术重构与优化 (Technical Polish)
*   **组件化重构**：将原本依赖于 `main.js` 全局函数的命令式逻辑（如角色详情渲染、搜索过滤）重构为纯正的 React 组件和 Hooks。
*   **状态管理**：利用 `zustand` 统一管理角色数据和搜索状态，使交互响应更加即时。
*   **性能优化**：优化图片加载，并确保动画流畅度。

## 实施步骤
1.  **基础建设**：安装动画库，配置全局 CSS 变量和主题切换。
2.  **导航与全局布局**：重塑 Navbar 和 Footer，添加交互元素。
3.  **首页华丽升级**：重点优化 Hero 区域和搜索交互。
4.  **角色详情页深度重构**：实现真正的数据驱动 UI，添加流畅的切换动画。
5.  **细节打磨**：全面检查交互反馈，确保移动端适配完美。
