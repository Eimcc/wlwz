# 虚拟人物日志网站 - 设计风格指南

## 设计理念

### 核心概念
- **内省的诗意**: 创造一个安静、温暖且富有思考性的数字空间
- **真实感与虚拟性的平衡**: 在数字化呈现中保持人物的真实性
- **时间的流动性**: 通过设计元素体现时间和天气的变化
- **情感共鸣**: 通过视觉语言传达角色的内心世界

### 颜色方案
**主色调**: 温暖的大地色系
- **主色**: `#8B7355` (温暖的棕色) - 代表稳重与温暖
- **辅助色**: `#A0937D` (浅棕灰色) - 用于次要信息
- **强调色**: `#D4B896` (奶油金色) - 用于高亮和交互
- **背景色**: `#F5F1EB` (米白色) - 主背景，营造纸质质感

**功能色彩**:
- **文本主色**: `#3A372E` (深棕灰) - 确保可读性
- **文本辅色**: `#6B6659` (中棕灰) - 次要文本
- **边框色**: `#E0DACE` (浅米色) - 分割线和边框
- **阴影色**: `rgba(139, 115, 85, 0.15)` - 柔和阴影

**天气主题色彩**:
- **晴天**: `#F4E4BC` (暖金色)
- **雨天**: `#B8C5D1` (灰蓝色)
- **阴天**: `#D4C4B0` (灰褐色)
- **雪天**: `#E8E0D6` (冷白色)

### 字体设计
**主标题字体**: Noto Serif SC (衬线体)
- 用于页面标题和重要标题
- 传达文学感和专业性
- 字重: 600 (SemiBold)

**正文字体**: Noto Sans SC (无衬线体)
- 用于日志内容和界面文本
- 确保长时间阅读的舒适性
- 字重: 400 (Regular), 500 (Medium)

**装饰字体**: ZCOOL XiaoWei
- 仅用于特殊标题和装饰性文字
- 增加手写感和亲和力

### 布局原则
**网格系统**: 12列响应式网格
**间距系统**: 基于8px的倍数
- 小间距: 8px, 16px
- 中间距: 24px, 32px
- 大间距: 48px, 64px

**页面结构**:
- **左侧边栏**: 1/4 宽度，角色信息面板
- **主内容区**: 2/4 宽度，日志显示区域
- **右侧边栏**: 1/4 宽度，时间和天气信息

## 视觉效果

### 使用的库和效果
1. **Anime.js**: 用于平滑的UI动画和过渡效果
2. **p5.js**: 创建动态背景粒子效果，模拟天气氛围
3. **ECharts.js**: 心情和活动统计图表
4. **Splide.js**: 历史日志轮播展示
5. **Matter.js**: 创建飘落的叶子或雪花物理效果
6. **Shader-park**: 创建柔和的背景渐变和光影效果
7. **PIXI.js**: 高性能的天气效果渲染

### 动画设计
**页面加载动画**:
- 角色信息面板从左侧滑入
- 日志内容淡入并向上移动
- 天气信息从右侧滑入

**交互反馈**:
- 按钮悬停: 轻微放大(1.05倍) + 阴影加深
- 卡片悬停: 向上移动4px + 阴影增强
- 链接悬停: 颜色渐变 + 下划线动画

**背景效果**:
- 使用p5.js创建微妙的粒子系统
- 粒子根据当前天气改变行为
- 颜色随时间变化(白天/夜晚)

### Hero区域设计
由于这是一个内省型的应用，不使用传统的Hero图像。取而代之的是:

**顶部状态栏**:
- 简洁的导航栏，包含Logo和页面切换
- 当前时间显示，字体优雅
- 实时天气图标和温度

**背景装饰**:
- 使用shader创建柔和的渐变背景
- 根据当前时间调整色调温度
- 添加微妙的纹理效果模拟纸张或布料

### 滚动动画
**日志条目动画**:
- 新日志从底部滑入，带有淡入效果
- 旧日志向上滚动，透明度逐渐降低
- 使用stagger效果创造层次感

**侧边栏动画**:
- 角色状态变化时使用颜色过渡动画
- 心情指示器使用呼吸动画效果
- 统计数字变化时使用计数动画

### 悬停效果
**角色信息卡片**:
- 悬停时轻微旋转(2度)
- 边框发光效果
- 内部信息显示更多细节

**日志条目**:
- 悬停时背景色变化
- 时间戳颜色变深
- 添加左侧装饰线

**天气图标**:
- 悬停时图标轻微摆动
- 显示详细的天气信息tooltip
- 背景色随天气类型变化

### 响应式设计
**桌面端 (1200px+)**:
- 三栏布局：角色信息 | 日志内容 | 天气状态
- 丰富的动画效果和交互

**平板端 (768px - 1199px)**:
- 两栏布局：角色信息 + 日志内容
- 天气信息整合到顶部
- 减少部分动画效果

**移动端 (< 768px)**:
- 单栏布局，垂直堆叠
- 角色信息可折叠
- 简化的动画，优化触摸体验

### 纹理和材质
**纸质纹理**:
- 在背景上添加微妙的纸张纹理
- 使用CSS滤镜创建柔和的噪点效果

**布料质感**:
- 在卡片和面板边缘添加柔和的阴影
- 模拟布料的柔软感

**水墨渲染**:
- 在装饰元素中使用水墨扩散效果
- 通过CSS渐变和透明度实现

### 图标设计
**天气图标**:
- 使用线性图标风格
- 配合颜色编码系统
- 添加微妙的动画效果

**状态图标**:
- 心情、活动、社交等状态使用简洁的符号
- 保持一致的设计语言
- 支持深色和浅色模式

### 数据可视化
**心情图表**:
- 使用柔和的曲线图
- 颜色渐变表示情绪强度
- 交互式数据点显示详细信息

**活动统计**:
- 使用环形图和柱状图
- 温暖的配色方案
- 平滑的动画过渡

**时间分布**:
- 使用热力图显示活动时间模式
- 颜色深浅表示活动强度
- 响应式布局适配不同屏幕

这个设计方案旨在创造一个既现代又温暖的数字体验，让用户在观察虚拟人物生活的过程中感受到平静和思考的氛围。通过精心选择的色彩、字体和动效，整个界面将传达出一种诗意和人性化的感觉。
