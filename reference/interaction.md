# 虚拟人物行为日志生成器 - 交互设计

## 核心交互功能

### 1. 虚拟人物信息面板
- 左侧固定面板显示角色基本信息
- 包含头像、姓名、年龄、职业、性格类型（MBTI/大五人格）、核心价值观
- 可点击展开查看详细的性格分析和背景故事
- 实时显示当前状态（心情、位置、活动）

### 2. 实时日志生成器
- 中央主要区域显示当前生成的行为日志
- 日志内容根据以下因素动态生成：
  - 当前时间（小时、星期、季节）
  - 新乡市实时天气（温度、天气状况、湿度、风力）
  - 角色性格特征和偏好
  - 历史行为模式
- 每30秒自动更新一次新的日志条目
- 包含时间戳、天气条件、行为描述、内心独白

### 3. 时间控制面板
- 右上角时间显示（模拟或真实时间）
- 季节和天气图标
- 可切换"实时模式"和"模拟模式"
- 模拟模式下可调节时间流速（1x、2x、5x、10x）

### 4. 天气信息展示
- 显示新乡市当前天气
- 温度、湿度、风力、空气质量
- 天气图标和背景色随天气变化
- 天气对角色行为的影响提示

### 5. 日志历史浏览器
- 可切换查看历史日志页面
- 按日期筛选日志条目
- 显示行为统计和模式分析
- 心情变化图表
- 最常出现的活动类型

### 6. 角色状态可视化
- 心情指示器（颜色编码的情感状态）
- 活动强度条（休息、轻度、中度、高强度）
- 社交需求指数
- 创造力水平指示器

## 用户交互流程

1. **首次访问**：用户看到角色基本信息面板和当前生成的第一条日志
2. **持续观察**：日志自动更新，用户观察角色行为变化
3. **深入了解**：点击查看角色详细信息和背景故事
4. **历史回顾**：切换到历史页面查看过往日志
5. **时间控制**：用户可选择加速时间观察不同时间段的行为模式

## 技术实现要点

- 使用天气API获取新乡市实时天气数据
- 基于角色性格矩阵和天气时间因子生成日志内容
- 使用本地存储保存历史日志数据
- 响应式设计适配不同设备屏幕
- 平滑的动画过渡增强沉浸感
