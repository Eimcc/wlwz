关于项目

探索虚拟人物的内心世界，理解技术与人性的交汇点

项目概述

这是一个独特的虚拟人物行为日志生成系统，通过模拟一个名为"林雨晴"的28岁咖啡馆店主的日常生活， 展现了技术如何用来探索和表达人性的复杂性。项目结合了心理学理论、行为建模和实时数据， 创造出一个既真实又富有诗意的数字生命体验。

"在数字时代，我们是否能够通过代码和算法来理解和表达人类的情感和内心世界？ 这个项目试图回答这个问题，不是为了替代真实的人际交往，而是为了更好地理解人性本身。"

林雨晴不仅仅是一个程序生成的虚拟角色，她是一个完整的、有深度的个体，拥有自己的性格特征、 价值观、情感状态和行为模式。她的行为会根据时间、天气、环境等多种因素动态变化， 展现出令人惊讶的真实感和一致性。

核心特性

深度角色建模

基于MBTI和大五人格理论构建的完整角色档案，包含详细的性格特征、核心价值观和行为偏好。

实时天气集成

实时获取河南省新乡市的天气数据，天气状况会影响角色的情绪和行为选择。

时间感知系统

角色具有完整的时间感知能力，不同时间段会表现出不同的行为模式和能量水平。

数据分析可视化

提供丰富的统计图表和数据分析功能，帮助理解角色的行为模式和情感变化趋势。

技术实现

前端技术栈

HTML5 CSS3 JavaScript ES6+ Tailwind CSS Anime.js ECharts.js p5.js

项目采用现代前端技术栈，确保了良好的性能和用户体验。Tailwind CSS提供了灵活的样式系统， Anime.js和p5.js则负责流畅的动画效果和视觉呈现。

核心算法

// 行为生成算法示例 function generateActivity(weather, time, personality) { // 基于时间和天气计算活动概率 let activityPool = calculateActivityProbability( time.getHours(), weather.condition, personality ); // 应用性格权重 activityPool = applyPersonalityWeights(activityPool, personality); // 加权随机选择 return weightedRandomSelect(activityPool); }

行为生成系统采用多层权重计算和概率选择算法，确保生成的行为既符合角色性格， 又具有一定的随机性和不可预测性，模拟真实人类行为的复杂性。

数据存储

使用浏览器的localStorage进行本地数据持久化，支持历史日志的查询和导出功能。 所有数据仅存储在用户本地，确保隐私安全。
